# Python Web Application with MySQL

This project demonstrates how to build a simple web application using Python and MySQL.

## Prerequisites

- Python 3.x
- MySQL Server

## Setup

1.  **Install Dependencies**
    ```bash
    pip install mysql-connector-python
    ```

2.  **Database Setup**
    Connect to your MySQL server and create a database:
    ```sql
    CREATE DATABASE python_web_sql;
    USE python_web_sql;
    ```

## Usage

Run the Python script:
```bash
python main.py
```

## Project Structure

- `main.py`: Main application entry point
- `README.md`: Project documentation
